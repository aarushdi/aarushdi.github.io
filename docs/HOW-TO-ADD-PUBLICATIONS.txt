===============================================
HOW TO ADD NEW PUBLICATIONS TO YOUR WEBSITE
===============================================

QUICK STEPS:
------------
1. Open publications/publications.bib in any text editor
2. Add your new BibTeX entry at the top or bottom
3. Commit and push.

That's it! There is NO build step. publications.html reads publications.bib
directly in the browser, so the entry appears automatically, sorted by year
with your name bolded.

publications.bib is the SINGLE SOURCE OF TRUTH. There is no generated data
file to keep in sync anymore (the old js/bibtex-data.js + update-bibtex.sh
have been removed). Just edit the .bib.

To preview locally before pushing, serve over HTTP (a browser will not let a
file:// page fetch the .bib):
    python3 -m http.server
    # then open http://localhost:8000/publications.html


EXAMPLE BIBTEX ENTRY:
---------------------

@article{rushdi2025example,
  author = {Ahmad A. Rushdi and Other Author and Another Author},
  title = {Your Paper Title Here},
  journal = {Journal Name},
  volume = {10},
  number = {3},
  pages = {100--120},
  year = {2025},
  url = {https://example.com/paper}
}


ENTRY TYPES YOU CAN USE:
------------------------

@article          - For journal papers
  Required: author, title, journal, year
  Optional: volume, number, pages, url

@inproceedings    - For conference papers
  Required: author, title, booktitle, year
  Optional: pages, publisher, url

@techreport       - For technical reports
  Required: author, title, institution, year
  Optional: address, url

@incollection     - For book chapters
  Required: author, title, booktitle, year
  Optional: pages, publisher, url


IMPORTANT FIELDS:
-----------------

author      Author names separated by " and "
            Example: Ahmad A. Rushdi and Jane Doe and John Smith
            (Your name will automatically be bolded)

title       Paper title

year        Publication year (used for sorting)

url         Link to the paper (optional but recommended)

journal     Journal name (for @article)

booktitle   Conference or book name (for @inproceedings, @incollection)

volume      Volume number (for journals)

number      Issue number (for journals)

pages       Page numbers (use -- for ranges, e.g., 100--120)

institution Institution name (for @techreport)

address     Location (for @techreport)

publisher   Publisher name (for books/chapters)


TIPS:
-----

• Publications are automatically sorted by year (newest first)
• Your name "Ahmad A. Rushdi" will be automatically bolded
• Use double hyphens (--) for page ranges: 100--120
• Always include the year - it's used for sorting
• Include URLs when available for easy access
• Use curly braces {} for field values


TROUBLESHOOTING:
----------------

Publications not showing?
→ Check for syntax errors in your entry (matching braces, commas, etc.)
→ If previewing locally, make sure you're using http://localhost (not a
  file:// path) — see the local-server command above.

Need more help?
→ See docs/README-BIBTEX.md for detailed documentation
→ Check js/bibtex-parser.js for formatting logic


FILES YOU SHOULD EDIT:
----------------------
✓ publications/publications.bib  - The ONE source of truth. Add entries here.

FILES YOU SHOULD NOT EDIT:
--------------------------
✗ publications.html      - Publications list is loaded dynamically from the .bib
✗ js/bibtex-parser.js    - Parsing/formatting logic (only touch to change layout)
