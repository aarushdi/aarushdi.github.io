Personal site — design mockups

Five minimal directions to pick from. Each is a standalone HTML file. Open one, scroll, click "Show more", resize the window.

Mockup 1

Editorial

Single narrow column, serif body (Source Serif), quiet maroon accent. Like a longform essay or a writer's homepage. No sidebar.

vibe: Karpathy / Cegłowski · serif body · top nav

Mockup 2

Academic Classic

Photo + name at top, tabbed nav under it, sans-serif body. Calm Stanford-blue accent, square avatar. The "expected" academic look, executed cleanly.

vibe: Jon Barron / Saining Xie · sans body · top photo

Mockup 3

Sidebar Minimal

Keeps the two-column structure of the current site, but strips gradients, animations, and colored badges. Same skeleton, much quieter.

vibe: keeps current layout · easiest content migration

Mockup 4

Single Page (anchored)

Sticky top nav, everything on one scrollable page with section anchors. Includes a 2×2 "Selected" preview grid linking to subpages.

vibe: modern docs / hashicorp · sticky nav · one URL

Mockup 5

Mono Zen

Monospace headings + dates, sans body, parchment background, single terracotta accent. Distinctive without being loud.

vibe: dev/hacker calm · monospace metadata · most opinionated

Mockup 6 — hybrid

Mono Zen + sticky top bar

Mockup 5's aesthetic with Mockup 4's sticky top nav. Photo hero, `[bracketed]` social links, dashed activity rows.

vibe: #5's mood + #4's navigation · the candidate

Mockup 6 — font lab

Floating picker bottom-right. Cycle through Current, Editorial (Fraunces × Amiri), IBM Plex, Geometric (Space Grotesk × Reem Kufi), Magazine (Instrument Serif × Aref Ruqaa), Scholarly (Newsreader × Markazi).

vibe: A/B test fonts side-by-side · choice persists across reloads
