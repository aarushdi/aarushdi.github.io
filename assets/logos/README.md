# Organization Logos

This folder contains small logo images used throughout the website.

## Required Logo Files

### Biography Section (Organization Logos)
These logos appear inline in the career history paragraph:

1. **sandia.png** - Sandia National Laboratories logo
2. **northrop-grumman.png** - Northrop Grumman Corporation logo
3. **ucdavis.png** - UC Davis logo
4. **utaustin.png** - UT Austin logo
5. **cisco.png** - Cisco Systems logo

### Recommended Specifications
- **Format**: PNG with transparent background (preferred) or SVG
- **Height**: 40-60px (will be scaled to fit text line height)
- **Width**: Auto (maintain aspect ratio)
- **Color**: Full color (original brand colors)
- **Background**: Transparent preferred

### Where to Find Logos

**Official Sources** (best quality):
- Sandia: https://www.sandia.gov/resources/
- Northrop Grumman: https://www.northropgrumman.com/newsroom/media-resources/
- UC Davis: https://ucdavis.edu/brand/downloads
- UT Austin: https://brand.utexas.edu/identity/signature/
- Cisco: https://www.cisco.com/c/en/us/about/brand-center.html

**Alternative Sources**:
- Wikipedia (Commons) - Search "[Organization name] logo"
- Wikimedia Commons - Often has official logos
- Brandfetch.com - Logo repository
- Organization's "Press Kit" or "Media Resources" pages

### Tips
- Prefer horizontal/wide versions of logos over stacked versions
- Ensure logos are high resolution (no pixelation)
- Use official brand assets when possible (avoids trademark issues)
- Test on both light and dark backgrounds if needed

### Installation
Once you have the logo files:
1. Save them to this folder with the exact filenames listed above
2. Refresh your browser (hard refresh: Cmd+Shift+R or Ctrl+Shift+R)
3. The logos should appear inline with the organization names

### Fallback
If you cannot find suitable logos, the site will show broken image icons, which you can style or replace with Font Awesome icons as a fallback.
